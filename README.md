Api Pokemon
=========

This API to list, create or buy pokemons.


## Technologies

* Gulp for compile task
* Docker
* Knex (Database Code First)

### Back-End
* Node
* Typescript
* Postgres (Knex + Objection.js) 

### Use
* list apps: localhost:3001/

## For Dev

```bash
# install depedencies
yarn

#run
docker-compose up
```
